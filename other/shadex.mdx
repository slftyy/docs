## Hiding Folder

<Warning>
  Make sure to disable your antivirus before using ShadeX.
</Warning>


<Steps>
  <Step title="First Step">
    Run ShadeX.
  </Step>
  <Step title="Second Step">
    Press `1` and `Enter` to Create a Folder.
  </Step>
  <Step title="Third Step">
    Now, a Folder named `Hidden` got created in your current directory.
  </Step>
  <Step title="Fourth Step">
    Now, leave ShadeX open and put all files that shouldnt be found, in this folder.
  </Step>
  <Step title="Fifth Step">
    Now, when your ready press `1` and `Enter` now the Folder is hidden, you can close ShadeX.
  </Step>
</Steps>
